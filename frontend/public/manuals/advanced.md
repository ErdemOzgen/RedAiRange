# Advanced Markdown Techniques

This document covers advanced markdown techniques including inserting code examples.

## Code Example in JavaScript

```javascript
function greet(name) {
    return `Hello, ${name}!`;
}

console.log(greet("World"));
```

## Code Example in Python

```python
def greet(name):
    return f"Hello, {name}!"

print(greet("World"))
```
